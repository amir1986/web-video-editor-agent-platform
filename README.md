# Web Video Editor Agent Platform

This is a local-first web video editor powered by an agent platform.

## Project Structure

- `apps/web`: Web UI with React + TypeScript + Vite
- `apps/api`: Gateway for AI interactions
- `packages/core`: Core data models and logic

## Implementation Plan

1. Create basic web app structure with 4 panels
2. Define ProjectState types in packages/core
3. Implement IndexedDB persistence
4. Add minimal styling
